# RSS-Namespaces

This focuses on RSS Namespaces in a podcast context. Some of these namespaces may also be observed in regular RSS feeds for blogs and similar.



The listed prefixes has been observed when parsing and indexing podcast feeds for [PodcastIndex.org](https://podcastindex.org/) and represent "known" prefixes, there is probably more out there that just hasn't been found yet.
